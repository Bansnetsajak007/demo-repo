        INITIALIZE THE SPECIFIC REPO

SECOND: 
        CREATE FILE, DO CODING AND EVERYTHING YOU WANT TO

THIRD: 
        AT BEGINNING THOSE FILES WILL BE AT UNTRACKED CONDITION

        TRACK THE FILE

        AFTER THAT COMMIT THE CHANGE USING SPECIFIC COMMIT MESSAGE
